## 温馨提示
- jupyter notebook 环境下，应使用 `uv pip install` 将依赖安装到 notebook 运行的虚拟环境下，并且保持 pyproject.toml 不改变。
  使用 `uv add` 安装依赖会导致 pyproject.toml 被更新。
